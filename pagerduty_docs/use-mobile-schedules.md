Use Mobile Schedules
====================

Review schedules and manage overrides in the mobile app

The PagerDuty mobile app is designed with a responder's experience in mind. You can easily access schedules, create overrides, and view previous and future on-call shifts.

> ðŸ“˜ Edit Schedules
> ----------------
> 
> If you would like to make structural changes to your schedules, such as adding/removing users from the rotation or adjusting shift lengths, you can do that in the web app. Please refer to [Edit Schedules](/main/docs/edit-schedules) for more information.

View Schedules
==============

To browse schedules in the mobile app, navigate to **More**  **Schedules**. You will be presented with two tabs: **My Teams** to view schedules associated with your [Teams](/main/docs/teams), and **All** to view all schedules in your organizationâ€™s account.

Select a schedule from the list to see its detailed view. A two-week calendar view is displayed at the top. Select a specific day or scroll through the timeline to find a particular shift. A shiftâ€™s duration is displayed for each entry, even if the shift spans multiple days.

Regular shifts will show the userâ€™s [schedule color](/main/docs/user-profile#schedule-color) in a solid color.

![A screenshot of the PagerDuty mobile app showing a regular on-call shift](https://files.readme.io/a7102de-image9.jpg "Regular shift")



Regular shift



Overridden shifts will display a horizontally striped color.

![A screenshot of the PagerDuty mobile app showing an overridden shift on a schedule](https://files.readme.io/6e141e1-image7.png "Override shift")

Overrides
=========

There are two ways to set an overrideâ€™s duration:

* [Enter Start and End Times](#enter-start-and-end-times)
* [Select a Duration](#select-a-duration)

Enter Start and End Times
-------------------------

### iOS

1. [Navigate to your preferred schedule](#view-schedules) and select the shift youâ€™d like to override.
2. If needed, tap **Starts** and **Ends** to modify the overrideâ€™s length. It will default to the remainder of the shiftâ€™s duration.
3. Tap **Override** in the top-right corner.

![A screenshot of the PagerDuty mobile app showing an override's start and end times](https://files.readme.io/7227a03-image2.png "Override start and end times")



Override start and end times




### Android

1. [Navigate to your preferred schedule](#view-schedules) and select the shift youâ€™d like to override.
2. If needed, to the right of the  icon, tap the first **date** and/or **time** to modify the start time, and the second **date** and/or **time** to modify the end time. It will default to the remainder of the shiftâ€™s duration.
3. Tap **Override** in the top-right corner.

![Override start and end times](https://files.readme.io/078c8f6a1413d70a4835796f092e255a42ddddfee6efb4400a2c3c32ee9f4034-mobile-schedules-override-time-android.png)



Override start and end times




Select a Duration
-----------------

### iOS

You can also modify an overrideâ€™s duration using a pre-populated option (e.g., **Until the rotation ends**).

1. [Navigate to your preferred schedule](#view-schedules) and select the shift youâ€™d like to override.

------------------

You can delete an override in the mobile app:

1. [Navigate to your preferred schedule](#view-schedules) and select the overridden shift.
2. In the **Shift Already Overridden** dialog, select **Delete Override**.
